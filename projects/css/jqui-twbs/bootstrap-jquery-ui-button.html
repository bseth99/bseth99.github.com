This page is isolated from the rest of the sandbox since it uses a newer Bootstrap and does not use the entire jQuery UI CSS Framework. I wrote a WaitButton widget which, by default, uses the jQuery UI styling and themes. However, now I use Bootstrap as my base framework and cherry-pick what I need from jQuery UI. For the WaitButton, it does everything I need but does not work with the button styles in Bootstrap. I used some of the work from addyosmani/jquery-ui-bootstrap but started with Bootstrap as the basis and simply integrated the necessary classes from jQuery UI Button to properly position the icon and button text. Since there are different color buttons, I created a spinner with a background to match each one. You can download the CSS by itself or zipped with the animated GIFs for the spinners.
